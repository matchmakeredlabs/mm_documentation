## GET - Decompose an Element Key into Component Descriptors

### Endpoint
```
https://matchmakeredlabs.net/api/key/{key}
```

Decompose an element Palet™ key into Component Descriptors.

---

### Request

#### Request Parameters

| Parameter | Type   | Required | Description |
|-----------|--------|----------|-------------|
| `key`     | string | Yes      | An element key made up of multiple intermediary statement keys. Example: `5001C-5073C-6200C-4955P-5247P-5358P-6350P-6736P-13396P`. |

### Response

#### Response Parameters

| Parameter     | Type   | Description |
|--------------|--------|-------------|
| `key`        | string | The element key consisting of multiple intermediary statement IDs. (`5001C-5073C-6200C-4955P-5247P-5358P-6350P-6736P-13396P`)|
| `statements` | array  | An array of statements with details on intermediary statements. |
| `id`       | string | The ID of the intermediary statement. |
| `rel`      | string | The relationship between the intermediary statement and the educational element. (`Central`, `Peripheral`) |
| `int`      | string | Intentionality between intermediary statement and the educational element. (`Intentional`, `Inferred`) |
| `stmtType` | string | The type of intermediary statement. (`STFk`, `CU`, `P`) |
| `statement`| string | The textual description of the intermediary statement. |
