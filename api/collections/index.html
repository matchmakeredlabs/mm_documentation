GET - All Collections​

Endpoint​

https://matchmakeredlabs.net/api/collections

Retrieve all collections.

Request​

Request Parameters​

None.

Sample Requests​

  • cURL
  • JavaScript
  • Python
curl \
-H "Authorization: Bearer $TOKEN" \
-X GET "https://matchmakeredlabs.net/api/collections"
let TOKEN = "YOUR_API_TOKEN"

      method: "GET", // *GET, POST, PUT, DELETE, etc.

          'Content-Type': 'application/x-www-form-urlencoded',
          'Authorization': `Bearer ${TOKEN}`
      },
    });
    return response.json(); // parses JSON response into native JavaScript objects

Response​

Response Parameters​

This GET request returns a list of all collections.

ParameterTypeDescription
eleTypestringType of element (any, c for curriculum, lr for learning resource, cs for competency statement, o for other)
namestringName of the descriptor. (Ex: "Grade 8: The Number System")
urlstringURL of the element.
aboutstringSubject of the element.
abstractstringAbstract describing the educational element.
identifierstringAssociated identifier. (Ex: "CCSS.MATH.CONTENT.8.NS.A.2")
educationalLevelstringGrade level. (Ex: "8")
creatorstringCreator of the element. (Ex: "Purplemath")
provenancestringThe person or entity that described an educational element. (Ex: "David Longdon")
keystringURL for associated key. (Each descriptor for an educational element is associated with multiple Palet™ statements, which make up a element key.)
mainEntitystringPrimary entity reference.
mainEntityIduuidUUID of mainEntity.

Response​

Response Parameters​

This GET request returns all descriptors in the collection specified by the ID. The descriptors are augmented by hierarchical information that assist the collection browsing.

ParameterTypeDescription
collectionIduuidUUID of the collection
collectionarrayAn array of descriptors containing the descriptors that are within the specified collection.
eleTypestringThe type of the educational element. (any, c for curriculum, lr for learning resource, cs for competency statement, o for other)
namestringName of the descriptor. (Ex: "Grade 8: The Number System")
urlstringURL of the element
aboutstringSubject of the element
abstractstringAbstract describing the educational element
identifierstringAssociated identifier. (Ex: "CCSS.MATH.CONTENT.8.NS.A.2")
educationalLevelstringGrade level. (Ex: "8")
creatorstringCreator of the element. (Ex: "Purplemath")
provenancestringDescriber of the element. (Ex: "David Longdon")
key stringThe URL for the associated element key
mainEntity stringIndicates the primary entity described in some page or other CreativeWork
mainEntityId UUIDUUID of a mainEntity
isPartOfstringURL of the descriptor's parent element
isPartOfIduuidUUID of isPartOf
intIdstringSequential ID (for internal use). A sequentially-assigned ID that's is assigned when you retrieve a collection and is only valid within that collection.
intHasPartarrayArray of intIds for child elements
rankstringHierarchical level. The level of a descriptor in the collection hierarchy. 0 is assigned to the top level item, 1 is assigned to children of that item, and so forth.
positionstringAn integer indicating the relative position of an descriptor among its siblings
leafCountstringThe number of leaf descriptors that are descendants (children, children of children, etc.) of a descriptor
leafWithKeyCountstringThe number of leaf descriptors that are descendants of a descriptor that have keys
