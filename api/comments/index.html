/comments

This endpoint allows you to create, get, update, and delete comments on Palet statements.

Table of Contents​

EndpointDescription
GET - All CommentsRetrieve all palet statement comments.
GET - Comment by IDRetrieve a specific comment by its ID.
POST - Create CommentCreate a new comment.
PUT - Update CommentUpdate a preexisting comment by its ID.
DELETEDelete a specific comment by its ID.

GET - All Comments​

Endpoint​

GET https://matchmakeredlabs.net/api/comments

Notes:

  • Returns all comments if caller has system-wide admin ReadDescriptor privilege (for now)

Request​

Request Parameters​

None required.

Sample Request​

  • cURL
  • JavaScript
  • Python
curl \
-H "Authorization: Bearer $TOKEN" \
-X GET "https://matchmakeredlabs.net/api/comments"
async function getComments() {
    const response = await fetch("https://matchmakeredlabs.net/api/comments", {
        method: "GET",
        headers: {
            'Authorization': `Bearer ${TOKEN}`
        }
    });
    return response.json();
}
