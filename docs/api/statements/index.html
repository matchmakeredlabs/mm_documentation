/statements

This endpoint provides access to Palet™ statements that describe educational elements.

Table of Contents​

EndpointDescription
GET - Lookup By IDRetrieve a Palet™ statement by its unique UUID.
GET - By KeywordsSearch for Palet™ statements using keyword-based search.

GET - Lookup By ID​

Endpoint​

https://matchmakeredlabs.net/api/statements/{statementId}

Retrieve a statement by its ID.

Request​

Request Parameters​

ParameterTypeRequiredDescription
statementIdstringYesUUID of the statement to retrieve (e.g., "2001-4830-9328-ff20").

Sample Requests​

  • cURL
  • JavaScript
  • Python
curl \
-H "Authorization: Bearer <your_token>" \
-H "Content-Type: application/json" \
-X GET "https://matchmakeredlabs.net/api/statements/2001-4830-9328-ff20"
async function getStatementById(statementId) {
    const response = await fetch(`https://matchmakeredlabs.net/api/statements/${statementId}`, {
      method: "GET",
      headers: {
          'Content-Type': 'application/json',
          'Authorization': `Bearer <your_token>`
      },
    });
    return response.json();
}

getStatementById("2001-4830-9328-ff20")
    .then((data) => console.log(data));
import requests

token = "<your_token>"
statement_id = "2001-4830-9328-ff20"
url = f"https://matchmakeredlabs.net/api/statements/{statement_id}"
headers = {
  'Content-Type': 'application/json',
  'Authorization': f"Bearer {token}"
}

response = requests.get(url, headers=headers)
print(response.json())

Response​

Response Parameters​

ParameterTypeDescription
getidstringID for intermediary statement.
statementsarrayAn array of statements (most likely of length 1) with the given statement ID
idstringID for intermediary statement.
stmtTypestringType of the intermediary statement. (STFk, CU, P)
statementstringThe intermediary statement.
kbstringKnowledge base name (subject).
kbidnumberKnowledge base ID.

Example Response​

{
  "getid": "2001-4830-9328-ff20",
  "statements": [
    {
      "id": "2001-4830-9328-ff20",
      "stmtType": "STFk",
      "statement": "Phase diagrams are graphic expressions of the relationship between pressure and temperature showing the conditions under which a substance will be in a particular state.",
      "kb": "Sci",
      "kbid": 1
    }
  ]
}

GET - By Keywords​

Endpoint​

https://matchmakeredlabs.net/api/statements?keywords={keywords}

Retrieve a statement by keywords.

Request​

Request Parameters​

ParameterTypeRequiredDescription
keywordsstringYesURI-encoded string of keywords for search (max 5 results returned).

Sample Requests​

  • cURL
  • JavaScript
  • Python
curl \
-H "Authorization: Bearer <your_token>" \
-H "Content-Type: application/json" \
-X GET "https://matchmakeredlabs.net/api/statements?keywords=water%20cycle"
async function getStatementByKeyword(keywords) {
    const response = await fetch(`https://matchmakeredlabs.net/api/statements?keywords=${encodeURIComponent(keywords)}`, {
      method: "GET",
      headers: {
          'Content-Type': 'application/json',
          'Authorization': `Bearer <your_token>`
      },
    });
    return response.json();
}

getStatementByKeyword("water cycle")
    .then((data) => console.log(data));
import requests
from urllib.parse import quote

token = "<your_token>"
keywords = "water cycle"
url = f'https://matchmakeredlabs.net/api/statements?keywords={quote(keywords)}'
headers = {
  'Content-Type': 'application/json',
  'Authorization': f"Bearer {token}"
}

response = requests.get(url, headers=headers)
print(response.json())

Response​

Response Parameters​

ParameterTypeDescription
keywordsstringThe search keywords used.
statementsarrayAn array of matching statements (max 5 results).
idstringID for intermediary statement.
stmtTypestringType of the intermediary statement. (STFk, CU, P)
statementstringThe intermediary statement.
kbstringKnowledge base name (subject).
kbidnumberKnowledge base ID.
